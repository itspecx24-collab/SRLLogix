Accessibility Statement

SRL-LOGIX is committed to digital accessibility for all users

Our Commitment to Accessibility

SRL-LOGIX is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

WCAG 2.1 Compliance

We aim to conform to level AA of the World Wide Web Consortium (W3C) Web Content Accessibility Guidelines (WCAG) 2.1. These guidelines explain how to make web content more accessible to people with disabilities.

Accessibility Features

  • Semantic HTML: Proper heading structure and semantic markup for screen readers
  • Alt Text: All meaningful images have descriptive alternative text
  • Keyboard Navigation: Full keyboard accessibility for all interactive elements
  • Color Contrast: Sufficient color contrast ratios for text readability
  • Responsive Design: Optimized for various screen sizes and devices
  • Focus Indicators: Clear visual focus states for keyboard users
  • ARIA Labels: Accessible Rich Internet Applications for dynamic content
